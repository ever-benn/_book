Section1.1

Section1.1

(1)

一开头，我的朋友，你的朋友，你的朋友胡先生几年前记下来了，早上在回到过去的梦里面醒过来这件事情，这样“从别后，忆相逢”的荒诞故事，后来成了他生活中的保留节目。我记得我来德国的时候还跟自己讲，休对古人思故国，且将新火试新茶，只是我是假的，你却是真的。

我不知道怎么我后来就不停的跟别人提起你来你，不由自主地我就会说起来，我好想念前任。我记得有一次在公园玩，我们坐在草坪上很开心很放松，我就跟朋友讲，我有一个朋友，曾经他真的很幼稚，即使他现在也好怀念他的前任。我朋友她坐在对面的人笑盈盈的就看着我说，这个人就是你吧？

我一直觉得，这个“这个朋友就是你的故事”更加适合做一个开头，因为这是我想给你看的，并不是什么那么痴情的姿态，而是一种，有一些浑沌的、矛盾的，想放下又不要放下的状态。

更何况其实你的朋友我，我也并不是那么三贞九烈的人，我也到底是分手之手追求过别人，我记得我们两个人捧着咖啡眼望着远方，好看的夕阳下面我没有说什么浪漫的话，我只是照旧絮絮叨叨的重复，我好想念前任啊。直到很久以后我才发现她听我一直在讲这些的时候是怎样的一种关爱智障的眼神在看着我。

(2)

好像你没有我的微博，那一年我还在微博上发了一张你的照片，配了那一句“昨天我数到，二十五颗星星，在北京，第二十五颗星星的夜晚”，感慨你要二十五岁了。我刚刚还写过，我有追过一个女生来着，我还有一次跟那个女生喝咖啡的时候不知道为什么就给她看了这条微博，我也不知道为什么，虽说事后我就只觉得自己是一个神经病一样。但是我还是在一直做这样子的事情。可是我还是在做这样的事情，我会把之前对面的的人关爱智障的微表情忘得一干二净然后重新体验一次尴尬。

我不知道你对我的印象还剩下多少，我总觉得你会很快忘了我，从最开始的一些细节开始，到我的外貌，最后我们的种种故事，我总觉得你很快都会把我彻底忘了。

只是倘若你还对我有一点点的印象，我觉得你一定会觉得这样的事情在这样的我身上发生，是一件合理的事情。

#

我不知道这个题目，你是不是会觉得很俗气，可是我实在是没想到更合适的，这是陆游的《沈园二首》，比起来“红酥手，黄縢酒，满城春色宫墙柳”那一首，我总觉得陆游四十年之后写的这一句“曾是惊鸿照影来”更加的有共鸣，没有那么那么矫情那么“艳”。

你记不记得，那一年你跟我说你要跟别的人在一起的时候我给你写的，“柳花夙有何冤业，萍末相遭乃尔奇”，“直到化泥方是聚，只今堕水未成离”。我很喜欢那一首因为我觉得“萍末相遭”合了萍字和远字，我一直相信你跟我在某一个维度上还没有分离。

我一直在挣扎着让自己相信这件事，我不想放弃。

#

但我其实也没有想的那么惨了，也不是说你一直是我生活生活的主旋律，只是说像之前说的，你一直存在在我的生活里面，以一种影影绰绰的样子存在着。

曾经我觉得那一段过去，是一件不能忘记的事情，是一个那么重要的存在，一年一年的过去，我心里面给自己许下的许多滥俗的辞藻，也渐渐的袪魅。于是我把那样子的话删了一些，不然你看到了，肯定会心里面觉得尴尬吧。

许多事情，那时候我无奈不已，与为什么我们要分开的任何一件小事情，都让我激动不已。我临走的时候跟你说话，总是歇斯底里的，不知道你还记不记得。只有这许多年过去之后，我这个幼稚的孩子才终于明白，我不应该这个样子，可是即使我现在懂了，如果让我再穿越回到从前，我还是觉得我不能控制自己的情绪。只是说我之前说的，“袪魅”之后，我能明白我所陶醉和无法自拔的情绪，并不是什么崇高和深刻的“爱情”，我歇斯底里，可能一大半就是因为我只是歇斯底里。

#

我真的是喜欢极了我走的时候你发的那一句，

“从我与他相熟，
我知道他的内心住着一个少年，
他是我的少年”。

我喜欢这句话，我爱极了这句话，我觉得我为这句话去死也死而无憾了。

#

我原来很开心，我在你这里还是有一些东西留了下来，有那样一些影响，是你总也抹不掉的。可是后来好像关于我的存在，你越来越来多的删掉了。梅香街十号，这不是我起的名字吗，怎么一看，不是这个名字了呢。怎么名字换了呢，不是说好的搬家了也不改名吗？

我说我看到你还是在放热河，心里一股无名火起，我说你看到你还转发了倪一宁的微博，心里面一阵冷笑，你对这些会觉得很不爽觉得很无语的对吧。你最后一篇更新关于武林外传，我心里面觉得写的不太行，很不太行，感觉跟以前知乎上好多人用这个套子也在写。你不用生气，我自己也是苦笑一声，我还是没有改啊，我这臭毛病，真的是个臭毛病呢。

你记不记得你有一次发了公众号我特别歇斯底里的给你打过电话去，发了一通脾气，说你不要在我的圈子里做这样的事情。现在我不会了，我看你发的东西，我也不会再激动，不会再跟你说话，也没有办法跟你说话了。 还是说回来吧，时间，时间，时间过得好快啊，你还在听热河吗，你还在想李志吗，你还记得他吗，他被封杀也已经过去了一年了。

#

我临出国的时候也开了一个公众号，写了一些尴尬的不痛不痒无病呻吟的东西，你记得吗，写什么我好想念你，什么我不出国了我回去找你吧什么的，令人尴尬的东西。后来我也什么都没有再写，再后来被你取关了，“胡叹叹啊”也改了名字。

再后来我也删掉了那些东西重新开始当qq空间写东西，又过了很长的时间，我有一个关系还不错的网友，她看了我的公众号，她跟我说我公众号以前的名字“胡叹叹啊”好可爱，我稍微吓到了一下，她怎么知道的。后来我发现，原来公众号的改名历史是可以看到的，我当时觉得好开心，我觉得“梅香街十号”这个名字虽然再也搜不到了，可是你看账号信息，那个名字还是在的。那感觉，就像是，你跟他相熟，他心里有一个少年，他是你的少年一样。觉得开心，我还在你那里，有一句肯定。

#

我真的特别好奇，你看到我写的东西，到底是什么样的感觉。可是只有一点就是，我并不想说我走出来了，我在大量的表达一种想要自暴自弃的情绪，大量的表达后悔和不甘心的情绪。虽然很多情绪我后来确实能理性的看待自己了，可是我没有想去开始新的生活。

#

多年以后我在网易云又翻出来了《秀水街》，你还记得我原来在公众号里面发的吗，秀水街上，天空晴朗，来往的人们，留着幸福的眼泪。我跟我朋友说，我在想，幸福的眼泪是什么感觉，我说我前任以前发过一回：听说一个人的苦难和快乐是恒定的，以后的生活，应该会很快乐吧。我说这可能就是流着幸福的眼泪吧。

我记得我跟你难过了一番，你感慨了两句“往者不可谏，来者要去追”，我还为此大为光火，你是不撞南墙不回头的人，你现在会觉得幸福吗，会觉得比我好吗？会的吧。

我记得你说你那一位答应你之后带你去厦门玩儿，厦门好玩儿吗？我也去了厦门，厦门确实很漂亮。我们各奔东西以后我也去了许多地方，乌镇的乌篷船头，周庄的水乡，澳门的许多粤式地名我不会念。科隆的大教堂很好看，柏林很穷很性感，马德里的街道很宽阔，新宿的夜生活很有趣，涩谷的衣服很酷，京都的枫叶特别好看，布鲁塞尔的啤酒特别好喝。我出国的时候跟自己说，“休对故人思故国，且将新火试新茶”，可是基本没有做到，我走到哪里也会想起你来。你记不记得有一年冬天我突然给你打电话说要给你寄明信片你当场挂了电话，那时候我在布鲁塞尔的撒尿小童旁边来着，哎，现在想想那时候我真冒昧啊。

#

我觉得我很努力，很多缺点都改了，有一次跟我同学聊天，他说不是的，我其实没有那么努力，不然不可能体型都控制不了。可能是的吧，许多缺点我都没有变，这些年好像离开你之后反而变本加厉了。

你很了解我吧，你深知我和你的不合适，可是你还是包容了我三年。我很让你失望吧，我是那么的不懂事。或许你也曾经鼓起勇气要爱我，可是我总是五分钟之后就会让你失望，可能我终究不是你值得爱的人。“男孩子的幼稚和女孩子的悲观，就是爱情的悲剧”，我常会想起来。我总是满嘴毒鸡汤，但是好像并不能做到什么。

许多次我觉得可能我不要结婚了，不要谈恋爱了，像我这样的人，始终就像一个只会读书的书呆子把，谢谢你让我看到了许多我从来没有想过的看过的东西。我记得后来认识许多人，许多事情他们都会说，没想到你抽烟，没想到你这么喜欢喝酒。他们总是说，你看起来就像一个乖小孩。乖小孩，这话反反复复的在出现。即使我讨厌我始终是这个样子，可是我好像也不太能改变的了我了。

#

要不，说说现在吧，说现在的事情。

谁能想到呢，我后来虽然没去你们企业，可还真是去了车企。

说来也有意思，我还经常关注你司的公众号呢，每年都会搜个一两次你们造的电车什么时候上市，有时候看照片还吐槽一下前脸内饰多少有点像友商。那时候我们还说要不以后一起去苏州，你肯定是忘了吧，我们那时候也只是很不认真的随便一说。我原来对汽车不感冒，还是觉得想做机器人，但是实在是没想到几年之后去大陆实习去了。

#

不知道你听过没有一个句式，“XX去了XX，XX去了XX，XX去了XX：他们都有美好的前程”。去日本的时候我也在想，大陆在上海有一家蛮大的分公司，我去了车企，你的舍友也去了车企，虽说你还在不在车企。总归是： 毕业之后，小米姑娘去北京造新能源汽车，她的好舍友去广州做混合动力汽车，他的好朋友去了上海汽车供应链企业：他们也都有美好的未来。

这真的很适合一些人青春生活的收尾，毕竟在南京的那四年，跟你在一起的那三年，是我在南京过得最开心，最自在，最舒服的时候。

不知道你记不记得我们刚在一起的时候在南农住了几天，你在我的桌面上新建了一个word文件，写的是“好奇大于爱情，不知道以后能否想拥，但是现在祝幸福。”这些年我一直保留着这个文件。我多怀念那时候的岁月，不管什么会议都有轻松明快的滤镜，我多喜欢那滤镜，那气氛，我多恨自己身在福中不知福一点也不知道珍惜。

写下来的时候在想，小米姑娘，多么遥远的名字啊，现在小米姑娘是不是已经成了小马姑娘了。你还会记得你用小米2的时候吗，那时候我在用魅族2的时候，心里面偷偷的觉得我们有种美好的对仗。后来你也不叫小米姑娘了，改成小码姑娘了，“胖子”、“大白”种种的称呼，后来成了“老胡”，后来再也没有了名字。

#

实习的时候，我在朋友圈发过一条，“顺其自然的回答，你要喝咖啡还是泡茶，下班之后，开心的去哪，我是真的不想回家”。我很喜欢这个朋友圈，我还记得这是横滨中华街，长长的扶梯上，我拍了一张照片发的那条朋友圈。

工作的时候真的会觉得疲惫、无聊、累，大家，包括你，都在说上班之后一瞬间整个人都会变，思维方式，生活方式，交友方式都会变，三年之后我才真的去体会，原来你说的是这样的。

我在Continental实习的时候，后半段搬到了横滨住，去附近的电车站每天要走一段不短的路。有一天回家的时候看谷歌地图，发现附近还有神奈川大学的学生宿舍在这里，我心里面吐槽了一句，他们出门做电车也要走很远的路啊。后来又一想，那时候我在读大学的时候，我们也有漫长的三号路呀，四年呀，那时候天天用双脚丈量三号路的长度。

不，我不是想说三号路的，那时候还是有时候会骑自行车去的。我是想说，我们常常连三号路都走不出去，就在半路吵起来，然后不欢而散了。我们就是这样神奇的一段感情啊。说起来自行车，你记不记得，《恋爱不是会骑自行车就行》，“而我和大白，即使我把自己练的可以得块儿奥运金牌，我们也不会好。说到底，谈恋爱和骑自行车有什么关系，那不过是锦上添花可有可无的东西。即使我舍友今天不会骑，他们仍然可以牵着小手儿漫步整个南京城，照样浪漫多姿。”而我们，就像是“两条垂直相交的射线”，只能卡在那里死磕。

虽说吧，你现在是天天的转发什么妈妈产后什么的，但是也还是能看到一点点你以前发的东西，我还是喜欢你以前发的东西，虽说好像我说的那一篇就没有了。哈哈。

#

我还是常常的梦到你，或许后面还会反复的说我梦到你的时候。有时候会梦到你去南京工作了，有时候会梦到我又回到了北京，梦到你来德国读书了，梦到你在日本旅游，梦到你在下北泽逛街，梦到你出轨，梦到我跟你抱在一起，我跟你又紧紧的抱在一起，我们相互许诺我们将会有一个美好的未来。梦到有人推门而入梦到你消失又不见。

我几乎不在11点之前躺在床上，不然我总是会想起你。说的不好听一些，浮躁，说的矫情一点，总是要把自己弄得筋疲力尽的才能睡着。才会想，原来失眠是这样的一种感觉，原来你之前的时候反复失眠是这样一种感觉。很不可置信吧。“夜阑卧听风吹雨，铁马是你，冰河也是你”。这怎么看都像是吹嘘吧。可是这对我越来越像是什么习以为常的事情，好像要伴随我许多年一样。

哪怕是因为我没有什么别人可以想呢。哪怕是，因为没有什么别人喜欢我呢。总之结果就是，你就像始终在我身边，没有走远一样。

我始终觉得我还在2016年，没有走多远，好像你还没有离开我多久一样。我每年都在说祝大家鸡年大吉，好像我们去看李志老师的演唱会还在没多久以前。

#

真的是好多年，我不知道我的朋友们会不会也有一个地方，写给某一个人倾诉心声，或许是再也见不到的前任，或许是再也见不到的亲人。这三四年，每每一个人的时候，常会写点什么东西，虽说统计统计总共也没有太多，也没有什么太杰出的段落。马东还能写个什么“伤情最是晚凉天”，我真是什么都写不出来。

这些年的心态起起伏伏，变来变去，这里面的情绪也一直变来变去，一开始写了一个虚构一个极为恬淡的生活，后来又一个极其愤怒、极其歇斯底里的开头。每一次都没有写下去，只是不断的往草稿里面填充想法。

#

我周围的朋友们好像都不怎么喜欢李志。可是我还是很喜欢，即使已经不听了，已经听腻了，可是我也还是常想起来他的音乐。突然就想起来你说你要跟你喜欢的人在一起的时候，“这次你离开了没有像以前那样说再见”，“我们从来没有想象的那么接近，只是两棵树的距离”，“南京的雨不停的下不停的下”。你记得三号路吗，三号路上的梧桐树，春夏的时候路上总会有一段时间梧桐絮漫天飞舞。所以“一转眼我们的城市又到了夏天，对面走来的人都眯着眼”。

我不知道怎么形容，总是反复的想起你来，一个个清晨和夜晚，总是反复的想看到你，想让你出现在我眼前。那种感觉从哪里来，不管一个人一个人来的走的，我总是在想，你如果在的话，会是什么样子，你在的话，会说什么话。

#

要不说点沉重的，之前读顾城和谢烨的故事，他们经过一段美丽的开始，然后相爱，可是之后顾城的偏执、特立独行等等特质，让他们的关系陷入了无限的痛苦。后来经过反反复复的种种痛苦，种种折磨，顾城和谢烨关系彻底崩溃，谢烨和英儿都离开了顾城，然后他举起斧头杀死了谢烨，然后自杀。

这个故事很有名了，由于只是几十年前的故事，有大量的资料留下来，顾城内心的纠葛、痛苦，顾城惊人的才华，顾城差劲的自理能力。

看完故事唏嘘一阵，我回忆了一下你，诶呀，你说要是你在读这个故事，会不会觉得你早早的离开我是一个明智的决定。会不会觉得有些庆幸，从前结婚、成家是明智的选择。读顾城留下来的文字，我是那么的沉浸在顾城留下来的语句里面，我觉得我还真有点理解他。诶呀，太难了。

#

我有时候看一些疯狂的男女关系结局，江歌刘鑫，被前夫烧死的拉姆。我看到的这些新闻有时候会想，你会不会想起来我暴躁的那些时候。是不是也会害怕我做出恐怖的事情。我打过你，你打过我，很遥远了吧，我们都记不清楚从前的种种口角是为什么。我只记得我有一种歇斯底里的情绪，好像是懦弱和暴躁，总是一种相互伴生的情绪。

所以看到许多新闻的时候我在想，你会不会觉得庆幸，没有跟我在一起。这种自卑在许多年里，一直缠绕着我。好像我跟别人都不能走太近，好像我并不会处理人和人的关系，好像我不能和别人好好相处。

当我想到自己没有钱的时候，当我想到自己前途不定的时候，当我看到自己日渐发胖，当我发现自己情商不够，种种的场景都能让我怀疑自己，是不是我不应该跟你在一起，是不是你离开我是明智的选择？ 哎呀，哎呀，我不好，我懦弱，我没有用吗。

#

我其实是没有想到，你会拿不同的男生们做比较。在我们最后的几次联系的时候，你说我不如那个人的时候，那么自然的说出来这句话。而且从你要跟别人谈恋爱到我们失联短短那么几天，也不只跟我说了一回。

哎呀，比如想起来这句话，我就还是有砸东西、发火的冲动。恨不得打一架，也恨不得从十二楼跳下去。这话我是如此的在意，我真的想不到，你居然会跟我讲这样的话，作这一种的比较。这个话即是如此的伤人，又是如此的违背我的三观。我是真的从没有想过把比较这个好那个不好这样的事情用在我身上了，我以为不管是朋友还是前任，这些人都不是被比较的，我觉得大家都会这样子，这委实是让我不曾想到。

#

之前跟朋友看《隐秘的角落》，看张东升突然我又看得“上头”了。怪不得大家那么喜欢秦昊演的张东生，“这个世界上不会再有像我一样对你好的人了。。。。我现在除了你什么都没有了，只要你不离开我我什么都可以接受。。。你们有没有特别重要的人，为了他们做一些自己不想做的事情。”我又激动了，我又情不自禁的把我自己带进去了，我觉得好多事情，真是我这样的人有可能做的事情。虽说我做不到对你那么好，这是这不妨碍其他的部分——我会激动到甚至去做很多违法乱纪的事情。

#

我是如此的疯狂。有一次在厨房跟舍友，一个女生，大吵了一架，事后我才发现我手上一直拿着菜刀。我朋友说起来她男朋友，会说起来两个人很开心，但是之后轻声的说，有点害怕的语气，“可是他有时候打我”。 我听到这话的时候脑海里面如此有画面感。好像我能把我自己代入进去。你对我模糊了吗？我是什么样的人？懦弱的，幼稚的，也有点理想主义的，我。

#

我不知道我是不是要这么急着讲一些辣眼睛的东西。嗯，还是讲的迂腐一点吧。

《冰与火之歌》里面，劳勃跟瑟曦结婚的第一天晚上，劳勃醉醺醺的跟瑟曦滚起了床单，结果兴头到了大喊了别的女生的名字。瑟曦抓狂了。

虽然我不是那么辣眼睛的人，不过我还是不太说的出口，那么荒谬的事情。
