Chapter1

我以前看一个陌生女人的来信，觉得不太接受。但是写到一半的时候回头看，可能这也像是“一封陌生男子的来信”。但是男生和女生的生活，毕竟是不一样的。但是，它甚至无法用“一封陌生男子的来信”这样暧昧不清的词语来粉饰，显得自己好像一往情深的样子一样。毕竟一个人去纠缠他的前任，且不说他有没有出息，纠缠许多年并不能证明它多么爱前任，说不定就只是他过得不好而已。真是奇妙，茨威格的小说大家都喜欢，可是当我摆出一副一往情深的姿态，我显得这么自私。

我大概平均一年会侧面打听一次你后来是什么样的生活，你且先不要想这些文字是否适宜送到你这里。这实在是一封不足为外人道、也未必适宜的信，我并不能证明它为什么值得你看，但我会觉得它不是不值得看的，毕竟是我自己写的嘛。但是他对我的意义，就是如果说死前想把什么事情做完的话，我原来觉得，死前写完它，不管是老死的还是自杀的，现在觉得，我应该这两年就会死去吧，只是就在这两年我就会死掉了，就这两年把它写完吧。

正如之前说的，我来德国的那一年，有一天我听到宋东野唱，“昨天我数到第二十五颗星星，在北京第二十五个秋天的夜晚”。那时候你快要过二十五岁的生日了，我心里面动了写点东西的念头。拖了这么多年，还是没有写好一个东西，我曾经设想过这个东西它很简短，只是在你二十五岁的生日之前的一封信，但是它拖了许多年；我一开始写的很恶心，很暴躁，它像某种无能狂怒。我写了一连串的无聊的反问句，既有骂你也有骂我，就像我们以前吵架一样的暴躁，也像以前吵架一样我一点也不见外，还是把你当作自己人；我设想过它会很宏大，很曲折，我做过一个设想，希望有机会能把他做成一个幽微曲折、剧情复杂的网页游戏，可是我创作不出来；我觉得它应该更有趣，更复杂，更值得一看，可是能力有限，我确实没有做到。

但是还是让我尽量维持一种轻松的气氛，再来讲一个故事吧。不知道是否你曾记得我有一年，在德国突然疯疯癫癫地告诉你，我过两天想回国见见你，去北京待一会儿就回德国。可是我终究也没有回去，2017年3月27号我离开亲爱的祖国，直到2019年4月1号才第一次回来。回来之后我找了附近的小店买了一张电话卡，然后就打车去了中航科技产业园，长城华冠距离机场真的好近啊，打个车十几分钟就到了。

不知道你是否会记得许多年前你曾接到过的一宗陌生电话，他某一个中午悄悄地出现在你的世界里面，毫无痕迹的出现又离开。那时候在门口犹豫了一会儿要不要给你打电话，犹豫来犹豫去，看了一会儿许多浩浩荡荡的队伍从那个大门走进走出，还是鼓足勇气用刚买的电话卡给你打了个电话。结果打过去之后你那边却毫无反应。。。。我跟我同学正聊微信来着，我跟她说没有反应，微信里面两个人是一阵一阵的“哈哈哈哈哈哈哈哈哈哈”，聊天从我紧张的情绪就变成幽默的气氛了。

在中航国际产业园那里，有一辆公交车叫空港五路，好像是去机场的吧，我当时候看到那个空港五路那个名字，突然想起来宋东野的《空港曲》。“但愿不在此时此地，但愿谁都不在意。”

结果在回山东的路上，电话打回来了。那时候我觉得真是很有意思了，我跟我朋友用更长的“哈哈哈哈哈哈哈哈哈”来表示更想笑了。我在回去的路上看看刚回来满身狼狈的自己，怂怂地没有接这通电话。

我之前的时候说，想见一见你，我当时生气说，明明从你们公司打车到机场好近，何止近，坐公交车也只要一会儿就到了,但是最后我们还是没有见面。我也是有很多小情绪的呀。因为走的时候终归是没有能见你一下，虽然这么近。你看，虽然我想尽量写得诙谐一点，但总归还是有很多小孩子的情绪吧，有一些小小的戾气在里面。不过这要比刚写的时候柔和的多了已经。嫉妒啊，嫉妒啊，十几分钟的路程，我们还是不能再见一面。我的朋友。

就像之前说的，第一次动这个念头，你二十五岁生日前夕那时候。我想起来，送你个什么呢，送你个自由吧。我消失掉，不打扰你了。

自由啊，挺有意思的词儿。

多年以后，我又听到了五月天在唱，“如果有，就让你自由，让你自由~ ~ ~ ~ ~”。

我现在还是记得第一次听这首歌是有一天在仙林那边，你还记得南京这个地理区域吗。有一天晚上我们在吵架，谁也不理谁，走在那边，走来走去谁也不说话。走到广场上，周围在放这首歌，突然我们就拥抱到了一起。我记得这首歌，五月天的《温柔》。“不知道不明了不想要，为什么，明明想要靠近，却孤单到黎明”，我记得就这一句印象好深啊，小姑娘，就记得那个晚上，五月天在唱这句话，反复地唱，我们那时候就突然拥抱到了一起。我好怀念那个时候啊，我已经记不起是哪里的广场了，金鹰广场，或者不在仙林在森林摩尔？你也不记得了吧。我只记得突然拥抱的时候那个感觉如此深刻，我们无数次吵架，无数次讨厌，无数次冷战，无数次谁也不理睬谁，也许多次突然就用抱在了一起，我们突然就和好了。

2021年8月我重新看到这里，我还是想到这样的事情会泪流满面。我们总是这样，漫长的冷战，又突然地、炽烈地和好，然后又是漫长的冷战、吵架、闹分手，在下马坊的地铁站前，在三号路上，在操场上，在校风碑那里，在你高中朋友拜访你的时候。我让你失望了，我给你丢脸了，我们不愉快了，但是我们又会突然地紧紧地拥抱在一起。往常怎么能只是往常，过去的东西怎么能只是过去，这么多年了，对，我还是不愿意忘记这些琐碎的东西，我也不愿意忘记你，我还是会觉得你是我生命里面最重要的人，不管失去什么，我愿意失去一切把你换回来。

睡不着的时候，等公交车的时候，抽烟发呆的时候，想起一点什么来就在这里写下来一两句，有空的时候就删删重复的。我想起来的过去，好像总是我们在闹分手。你总是想分手，我也总是闹分手，我们总是吵架，谁也不理谁，我们有时候很恩爱，很开心。这里对我来说，像是某种地层、年轮一样的东西。一层层的堆积，一点点的变化，有时候豪情万丈，有时候阴郁绝望，一会儿哭，一会儿笑，一会儿愤怒，一会儿绝望。

哦，对了，分手几年之后，我也开始抽烟了，这倒不是因为分手的原因，不过确实是生活有压力，我开始抽烟发呆来缓解压力了。虽然很滑稽，抽烟的时候经常也并不过肺，也并没有那么有感觉，只是在发呆，在排解而已。
